- O projeto The Virtual Pantry tem como principal objetivo facilitar a tua vida: estás fora de casa e não sabes se tens um ingrediente em casa necessário para fazeres uma receita que adoras? Agora já sabes se tens de ir às compras!


- Odeias fazer inventários à tua despensa? Este projeto é para ti! Basta manteres atualizados os teus "gastos" e a tua "ementa" e tens listas de compras automáticas!


- Não sabes onde guardar as tuas receitas favoritas? E que tal manter tudo no mesmo sítio? Assim sabes imediatamente se tens tudo o que é preciso!
